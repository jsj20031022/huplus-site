# HuPlus.asia 上线教程（15分钟搞定）

## 第一步：注册 Cloudflare 账号（3分钟）

1. 打开 https://dash.cloudflare.com/sign-up
2. 输入你的邮箱 + 设置密码，点击 Create Account
3. 验证邮箱（查收邮件，点击确认链接）

---

## 第二步：部署网站到 Cloudflare Pages（5分钟）

1. 登录后，左侧菜单点击 **"Workers & Pages"**
2. 点击 **"Create"** → 选择 **"Pages"**
3. 选择 **"Upload assets"**（直接上传文件，不需要 GitHub）
4. 项目名称填 `huplus` 或任意名字
5. 把整个 `huplus-site` 文件夹拖进上传框（或点击选择所有文件）
6. 点击 **"Deploy site"**
7. 等待约30秒，部署完成！你会得到一个类似 `huplus.pages.dev` 的免费链接，可以先访问确认效果

---

## 第三步：绑定你的域名 huplus.asia（5分钟）

### 3a. 把域名转入 Cloudflare（推荐，免费DNS）

1. 在 Cloudflare 控制台，点击左上角 **"+ Add a domain"**
2. 输入 `huplus.asia`，点 Continue
3. 选择 **Free 套餐**，点 Continue
4. Cloudflare 会给你2个 Nameserver（NS）地址，例如：
   - `ann.ns.cloudflare.com`
   - `bob.ns.cloudflare.com`
5. 登录你购买域名的注册商（GoDaddy / 阿里云 / Namesilo 等）
6. 找到 huplus.asia 的 DNS 设置，将 Nameserver 改成上面两个
7. 等待生效（通常10-30分钟，最长48小时）

### 3b. 绑定域名到 Pages

1. 回到 Cloudflare Pages → 你的 huplus 项目
2. 点击 **"Custom domains"** → **"Set up a custom domain"**
3. 输入 `huplus.asia`，确认
4. 自动配置 SSL 证书（免费，自动续期）

---

## 第四步：申请 Google AdSense（第2天起）

1. 打开 https://adsense.google.com
2. 注册账号，填入 `huplus.asia`
3. 将审核代码粘贴到 index.html 的 `<head>` 里（我会帮你操作）
4. 等待审核（通常1-2周，网站需要有一定内容）
